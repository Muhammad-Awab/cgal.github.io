template<typename GeomTraits, typename IntersectionTraits = CGAL::Exact_predicates_exact_constructions_kernel>
class CGAL::Kinetic_shape_partition_3< GeomTraits, IntersectionTraits >

creates the kinetic partition of the bounding box of the polygons given as input data.

Use Kinetic_shape_partition_3() to create an empty object, insert() to provide input data and initialize() to prepare the partition or use Kinetic_shape_partition_3()

Template Parameters
GeomTraitsmust be a model of KineticShapePartitionTraits_3.
IntersectionTraitsmust be a model of Kernel using exact computations. Defaults to CGAL::Exact_predicates_exact_constructions_kernel.
Examples:
Kinetic_shape_reconstruction/kinetic_precomputed_shapes.cpp, and Kinetic_shape_reconstruction/kinetic_random_shapes.cpp.
